# Pok Transfer

A simple commandline application that can transfer Pokemon between different versions of the handheld Pokemon games. As of now transfers between
Red, Blue, Gold, Silver, Fire Red and Leaf Green are supported.

Designed to transfer between generations. Designed to preserve traits like gender and shininess between generations.

This is not a tool to hack your save games. Things like pokemon generation and item manipulation is not supported.

# Installation

Requires Zig 0.15. Lower and higher versions of Zig are not compatible
